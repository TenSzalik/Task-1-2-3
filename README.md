3: "d" "e" "f",

4: "g" "h" "i",

5: "j" "k" "l",

6: "m" "n" "o",

7: "p" "q" "r" "s",

8: "t" "u" "v",

9: "w" "x" "y" "z",

Task 3:

Zwraca wyjustowany przez nas tekst w odpowiedniej szerokości (również przez nas podanej) w formie listy. Gotywy tekst ma równomierne spacje i jest wyrównany do lewej.
